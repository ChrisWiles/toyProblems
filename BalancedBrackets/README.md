## Balanced Brackets

Given a string, return true if it contains all balanced parenthesis 

(), curly-brackets 

{}, and square-brackets 

[].

#### Examples

InputOutput

 **str:**  
"(x + y) - (4)" true

 **str:**  
"(x + y) - (4)" true

 **str:**  
"(((10 ) ()) ((?)(:)))" true

 **str:**  
"[{()}]" true

 **str:**  
"(50)(" false

 **str:**  
"[{]}" false
