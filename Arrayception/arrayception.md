## Arrayception

Given an array of arbitrarily nested arrays, return 

n, where 

n is the deepest level that contains a non-array value.

#### Examples

InputOutput

 **array:**  
[ [ 5 ], [ [ ] ] ] 2

 **array:**  
[ 10, 20, 30, 40 ] 1

 **array:**  
[ [ 10, 20 ], [ [ 30, [ 40 ] ] ] ] 4

 **array:**  
[ ] 0

 **array:**  
[ [ [ ] ] ] 0
